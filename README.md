# Robot-Control-Algorithms-Simulator-Website-
The Robot Control Algorithms Simulator is a browser-based interactive application that simulates three fundamental robot control algorithms in real time using HTML5, CSS3, and JavaScript.  

Installation Manual
Robot Control Algorithms Simulator

What You Need

A computer running Windows, Mac, or Linux
A modern web browser such as Chrome, Firefox, or Edge
The project folder containing all 6 files


File Structure
Make sure your folder looks exactly like this before opening anything:
robot-control/
│
├── index.html
│
├── css/
│   └── style.css
│
└── js/
    ├── pid.js
    ├── statespace.js
    ├── trajectory.js
    └── main.js

Steps to Run
Step 1 : Download all the files from Canvas or wherever they were shared.
Step 2 : Create a folder on your Desktop called robot-control.
Step 3 : Inside that folder create two more folders, one named css and one named js.
Step 4 : Place index.html directly inside robot-control, place style.css inside the css folder, and place all four .js files inside the js folder.
Step 5 : Double click index.html and it will open automatically in your default browser.
Step 6 : Click any of the three tabs at the top to switch between PID Controller, State-Space, and Trajectory Tracking.
Step 7 : Press the RUN button to start the simulation and use the sliders on the left to tune the parameters in real time.

Important Notes

No internet connection is required after the page loads once (Chart.js loads from online on first open)
No Python, Node.js, or any software installation is needed
If the page looks broken, make sure the css and js folder names are spelled in lowercase exactly as shown above
If charts do not appear, try opening the file in Google Chrome instead
